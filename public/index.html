Toys API Documentation

Method Endpoint Description Request Body Fields Query Parameters
GET /toys Get a paginated list of toys - perPage (optional), page
GET /toys/search Search for toys based on a query string - s = (info or name)
GET /toys/category/:catName Get toys by category - catName (category name)
GET /toys/single/:id Get a single toy by ID - id (toy ID)
POST /toys Create a new toy name, info, category, img_url, price -
PUT /toys/:editId Edit a toy name, info, category, img_url, price -
DELETE /toys/:delId Delete a toy - -
